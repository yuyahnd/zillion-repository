# CAD
Command Assist Driver

## Install & venv

```
$ git clone https://github.com/yuyahnd/cad.git
$ python -m venv venv
$ . venv/Scripts/activate
$ python -m pip install --upgrade pip
$ pip install -e .
```

```
$ deactivate
```

## Test
```
$ pip install pytest
```

```
$ pytest
$ pytest -v --durations=0
```
